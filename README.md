# 股票形态选股系统

## 项目概述
基于模板股票日线图的形态识别，自动筛选A股市场中相似走势的股票。

## 目录结构
- `data/` - 股票数据存储
- `templates/` - 模板图片存储
- `uploads/` - 用户上传图片
- `results/` - 筛选结果
- `src/` - 源代码

## 使用方法
1. 在 `templates/` 目录上传模板股票日线图
2. 运行主程序进行全市场筛选
3. 查看 `results/` 目录获取匹配结果

## 特性
- ✅ 排除停牌、ST、亏损股票
- ✅ 每日收盘后自动更新
- ✅ 纯日线形态分析（无其他指标）
- ✅ 零费用运行
- ✅ GitHub协作支持
